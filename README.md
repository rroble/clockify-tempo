# Clockify Tempo


### log4Month

Create logs to tempo and clockify for the whole month.
Create and tracks runId temp file to avoid logging duplicates.

---

### Prepare env and mapping
1. Copy `.env.sample` to `.env` and update your personal details/secrets.
2. Update mapping.json


### Run
@see `log4Month.sh`
